# Методы массивов {#array-methods}

```js
const arr = [1, 2, 3, 4, 5];

arr.push(6); // добавляет элемент в конец массива

/*
 * Принимает функцию обратного вызова (callback)
 * Функция callback принимает 2 аргумента:
 * - текущий элемент
 * - индекс текущего элемента
 *
 * function (item, index) { return item }
 *
 * Должна возвращать элемент
 */
arr.map((item) => item * 2); // возвращает новый массив, в котором каждый элемент умножен на 2

/*
 * Принимает функцию обратного вызова (callback)
 * Функция callback принимает 2 аргумента:
 * - текущий элемент
 * - индекс текущего элемента
 *
 * function (item, index) { return boolean }
 *
 * Должна возвращать true или false
 */
arr.filter((item) => item % 2 === 0); // возвращает новый массив, в котором только четные элементы

/*
 * Принимает функцию обратного вызова (callback) и начальное значение
 * Функция callback принимает 2 аргумента:
 * - currentAccumulator - текущее значение аккумулятора
 * - item - текущий элемент
 *
 * .reduce(callback, initialValue)
 * function (currentAccumulator, item) { return newAccumulator }
 *
 * Функция callback должна возвращать новое значение аккумулятора
 * Второй аргумент reduce - начальное значение аккумулятора
 * Если не передать второй аргумент, то начальное значение будет равно первому элементу массива
 * */
arr.reduce((acc, item) => acc + item, 0); // возвращает сумму всех элементов массива
```
